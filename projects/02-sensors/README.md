# Sensor Projects

This folder contains projects involving various sensors.

## Common Sensor Types

### Environmental Sensors
- **DHT11/DHT22**: Temperature and humidity
- **LM35**: Temperature sensor
- **BMP180/BMP280**: Pressure and altitude
- **MQ Series**: Gas and smoke sensors

### Motion and Distance Sensors
- **HC-SR04**: Ultrasonic distance sensor
- **PIR**: Motion detection
- **MPU6050**: Accelerometer and gyroscope
- **HMC5883L**: Magnetometer/compass

### Light and Color Sensors
- **LDR**: Light dependent resistor
- **BH1750**: Digital light sensor
- **TCS3200**: Color sensor

### Other Sensors
- **Sound sensor**: Detect sound levels
- **Soil moisture sensor**: For plant monitoring
- **Rain sensor**: Detect water/rain
- **IR sensor**: Infrared detection

## Key Concepts

- Analog sensor reading and calibration
- Digital sensor protocols (I2C, SPI)
- Data filtering and smoothing
- Sensor libraries and documentation
- Threshold detection

## Wiring Tips

- Always check sensor voltage requirements (3.3V vs 5V)
- Use pull-up/pull-down resistors when needed
- Keep sensor wires short to reduce noise
- Use appropriate power supply for power-hungry sensors
